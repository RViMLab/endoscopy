# Endoscopy
Utilities for processing of endoscopic images.

## Installation
To install, simply run
```shell
pip install endoscopy
```

## Example
Check out the [examples](https://github.com/RViMLab/endoscopy), run
```shell
python img_main.py # vid_main.py
```
